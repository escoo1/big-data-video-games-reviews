# Big Data Video Games Reviews

Big Data Analytics course project using the Amazon Reviews 2023 Video_Games dataset.

## Team Members

- Bach Leno
- Escobar Rüsch Oliver
- Vilkin Zakhar

## Project Goal

Analyze customer review patterns in Amazon Video Games data using Apache Spark and MongoDB.

## Repository Structure

- `data/` raw and processed data
- `notebooks/` notebooks and experiments
- `src/` Python scripts
- `docs/` proposal and report files
- `results/` visualizations and outputs
